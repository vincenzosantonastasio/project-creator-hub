# Project Creator Hub

Organizing the creator economy around creators, not platforms.

---

## The Problem

Today, people follow creators across many platforms:

- YouTube  
- TikTok  
- Instagram  
- Podcasts  
- Blogs  
- Newsletters  

Content is fragmented.

Users have to jump between platforms to follow a single creator.

---

## The Idea

> People follow creators, not platforms.

Project Creator Hub is a platform designed to organize all content from a creator into one place.

---

## What the Platform Does

- Aggregate content from multiple platforms  
- Display everything in a single creator page  
- Allow users to follow creators  
- Show only new content since last visit (Catch-Up feature)  
- Help users discover new creators  

---

## Core Feature

### Creator Page

A single page where all content from a creator is organized and accessible.

---

### Catch-Up System

Users see only content published since their last visit.

This reduces overload and improves experience.

---

## Platform Philosophy

- Not a social network  
- Not a content hosting platform  
- Not a messaging system  

The platform organizes content and redirects to original sources.

---
